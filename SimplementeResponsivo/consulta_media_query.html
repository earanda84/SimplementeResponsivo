@media queries

Css uses the @media rule to detect information about the device being used to acces the website. For example, it can be uesed to detect the size of the device, its capabilities, and its orientation. For this course, we will be using it to detect the device size, but kepp in mind that it is capable of much more.

Phone: Width less than or equal to 480px.

Tablet: width between 481px and 1023px.

Desktop: Widh greater than or equal to 1024px.
